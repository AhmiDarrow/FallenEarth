# Dispatches (FallenEarth)

Self-contained task packages created automatically by Remedy.

- `claude/` : For Claude Code (or local equivalent). Contains DISPATCH_*.md with exact atomic task + files to read + strict "do only this + handoff back" rules.
- `hermes-sub/` : For internal Hermes sub-agent focus.

Remedy creates these (and the matching handoff + LATEST update) when it routes work out. The receiver loads the dispatch + latest handoff + CLAUDE.md/AGENTS.md and stays atomic.

Example dispatch files will appear here when used.
