## Key Design Decisions

1. **Colored rectangles for furniture** — Simple colored rectangles for each furniture type, with different colors for wood (brown), metal (grey), fire (red), straw (tan). Labels for signs/podiums.

2. **Furniture as collision** — Furniture blocks movement like walls. This prevents players from walking through tables, barrels, etc. Simple and effective.

3. **Data-driven furniture** — Furniture defined in JSON data, not hardcoded. Easy to add new furniture types or modify room layouts.

4. **11 furniture types** — Comprehensive set covering tavern (tables, barrels), shops (shelves, crates), blacksmith (anvil, forge, rack), arena (dummies, rack), and general (benches, signs, podiums).
