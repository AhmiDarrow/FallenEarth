## 3. Key Decisions & Rationale

- **Three-lane spawning** (not two): separates MobSpawner beasts, EncounterBuilder humanoids, and future RiftSpawner
- **`wildlife_class` as property flag** (not separate category file): keeps mobs.json flat, MobSpawner filters by comma-separated class list
- **SKITTISH state separate from FLEE**: FLEE is reactive (after damage), SKITTISH is proactive (none before player approach)
- **Placeholder sprites first**: PixelLab queue is slow (3/20 after ~15 min), placeholders let the game function immediately
- **size=180 for PixelLab**: avoids review step (≥171px = 1 candidate, auto-complete); existing mobs mix 68px and 180px sizes
