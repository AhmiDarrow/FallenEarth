## 2. Key Decisions & Rationale

| Decision | Rationale |
|----------|-----------|
| `TextureRect` with `STRETCH_KEEP_ASPECT_COVERED` | Preserves image aspect ratio while filling screen; standard for background art |
| `expand_mode = 1` (IGNORE_SIZE) | Allows texture to stretch beyond original size without affecting layout |
| ConfigFile for settings | Simple Godot API; stores in `user://settings.cfg`; human-readable |
| Monitor selection moves window | Godot lacks `set_monitor()`; moving window to screen position achieves same effect |
| Resolutions filtered by monitor size | Prevents selecting larger‑than‑monitor resolutions; common list covers most use‑cases |
| Settings applied immediately on change | Better UX; user sees result before clicking Apply (Apply still exists for explicit confirmation) |
