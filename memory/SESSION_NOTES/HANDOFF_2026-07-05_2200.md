### F1: `assets/sprites/ui/buttons/` — Button assets

### F2: `scripts/ButtonStyleHelper.gd` — Button style helper
- Helper class to apply button textures to UI buttons
- Provides static methods:
  - `apply_style(btn, style_key)` — apply any style
  - `apply_primary(btn)` — apply primary style
  - `apply_secondary(btn)` — apply secondary style
  - `apply_danger(btn)` — apply danger style
  - `apply_success(btn)` — apply success style
- Falls back to StyleBoxFlat if textures fail to load
- Sets appropriate font colors for each style

## Key Design Decisions

1. **PixelLab MCP for button generation** — Used PixelLab MCP to generate consistent pixel art button textures with post-apocalyptic sci-fi aesthetic.

2. **Four button styles** — Primary (main actions), Secondary (cancel/back), Danger (retreat/exit), Success (confirm/accept). Covers all UI needs.

3. **Texture + fallback approach** — ButtonStyleHelper tries to load textures first, falls back to StyleBoxFlat if textures fail. Ensures buttons always work.

4. **256x192px button size** — Large enough for text labels, small enough for UI panels. Consistent with game's pixel art style.
