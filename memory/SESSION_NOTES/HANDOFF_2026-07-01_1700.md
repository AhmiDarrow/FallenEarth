## 1. What Was Accomplished

- **Two-layer world model** — Planet hex sphere (strategic) + 512×512 local playfield per hex (tactical)

- **`WorldMapScreen.gd` + `scenes/WorldMapScreen.tscn`** — RimWorld-style strategic map: fog of war, faction ★, quest !, rift ⚡, adjacent-only travel

- **`GameState.gd` extended** — `hex_states`, `local_x/y`, `discovered_hexes`, `travel_to_hex()`, per-hex persistence; terrain stripped from save (regenerated from seed on load)

### Phase 5 — Rift loop + persistence
- **Rift local coordinates** — `RiftRunner` rifts have `local_x/local_y`; spawn/check at player cell
- **Return to entry** — `RiftInstance` restores `entry_local_x/y` after close or back; `CombatEncounterBuilder.build_rift_room()` passes local entry through combat round-trip
- **Save/load** — `overworld_mobs`, `rift_state` persisted via `GameState` → `SaveManager`; `RiftRunner.get_save_payload()` / `load_from_save()` / `reset_for_new_game()`

### Missions aligned to local map
- Mission combat mobs spawn at deterministic `target_local_x/y` within target hex
- `!` mission markers on local map; `build_mission_encounter()` uses full `q,r|lx,ly` mob keys
- `should_block_move_for_mission(hex_key, local_x, local_y)` for precise trigger

### Phase 6 — Performance
- **`LocalMapRenderer.gd`** — 32×32 cell chunk streaming; loads/unloads chunks around player viewport

### Validation

## 2. Key Decisions & Rationale

| Decision | Rationale |
|----------|-----------|
| Hex = region metadata; 512×512 = gameplay | Matches RimWorld world map vs colony map; scales content without simulating whole planet |
| 4 cardinal edges map to axial neighbors | Simple v1; diagonal hex neighbors via world map travel |
| Terrain not saved in `hex_states` | 512×512 PackedByteArray too heavy for JSON; deterministic regen from `local_seed` |
| Mob keys: `q,r\|local_x,local_y` | Sphere hex for travel; local coords for combat/rifts on tactical layer |
| Chunk size 32 cells | Balance between node count and reload frequency at 24px cells |
| Assets out of scope this session | Parallel agent on ComfyUI/hand-drawn tiles; code uses ColorRect terrain until overlay hook added |

See `docs/ARCHITECTURE.md` for updated flow diagram and layer descriptions.
